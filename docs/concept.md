# Concept

## Core

### Problem

Затратность тонкой настройки вебпака для каждого нового проекта.

## Solution

Создать библиотеку, которая предоставляет самые распространённые настройки по умолчанию, и так же может быть легко тонко настроена при помощи композируемых функций, каждая из которых реализует определённую часть конфига.

## Implementation

Экспортировать функцию, принимающую глобальные настройки конфигурации и возвращающую объект с фичами. Каждая фича регистрируется в системе и может быть расширена при помощи плагинов.
